# TOEFL B1 Day 3 学习计划

日期：2026-07-10  
总时长：2 小时  
Day 3 主题：听力细节，错因分类。

## 今日材料

- Sprout：选 1 个 B1 English native-speaker blog/video，优先选学习、工作、心理、健康、效率类主题。
- B1 听力：[British Council - An introduction to a lecture](https://learnenglish.britishcouncil.org/free-resources/listening/b1/introduction-lecture)
- B1 阅读：[British Council - Planning an event](https://learnenglish.britishcouncil.org/free-resources/reading/b1/planning-event)

## 今日目标

- [ ] 听力前先预测 5 个关键词。
- [ ] 听力做题只记录第一次答案。
- [ ] 每道错题写错因：没听到关键词 / 混淆选项 / 没看清题干 / 时间压力。
- [ ] 阅读继续找证据句，但今天阅读不加难度。

## 0:00-0:10 Day 2 错题复盘

- [ ] 看 Day 2 的 3/6 那组错题。
- [ ] 每道错题只写 1 个错因。
- [ ] 选一个最常见错因，写到今天标题下。
- [ ] 重听或重看错题附近 30 秒。
- [ ] 写今天提醒：Listen for keywords, not every word.

## 0:10-0:30 Sprout B1 泛听

- [ ] 选 1 个 B1 native-speaker blog/video。
- [ ] 第一遍不看字幕。
- [ ] 写 5 个听到的关键词。
- [ ] 写 1 句主旨：This blog is mainly about ...
- [ ] 给自己打分：主旨 0 / 1 / 2 / 3。

## 0:30-0:45 Sprout 精听

- [ ] 第二遍听，可以看字幕。
- [ ] 暂停跟读 5 句。
- [ ] 抄 3 个表示原因、步骤、结果的表达。
- [ ] 标出 2 个信号词：first / then / because / so / however。

## 0:45-1:15 B1 听力细节

材料：[An introduction to a lecture](https://learnenglish.britishcouncil.org/free-resources/listening/b1/introduction-lecture)

- [ ] 先看标题，预测话题。
- [ ] 做 preparation task。
- [ ] 听第一遍，不看 transcript。
- [ ] 写主旨：The lecture introduction is mainly about ...
- [ ] 做 Task 1 和 Task 2，只记录第一次答案。
- [ ] 看 transcript，标出 5 个细节信号：time / order / example / reason / contrast。
- [ ] 每道错题写错因。

## 1:15-1:40 B1 阅读细节

材料：[Planning an event](https://learnenglish.britishcouncil.org/free-resources/reading/b1/planning-event)

- [ ] 先看版式和人名。
- [ ] 圈出所有 person、task、time。
- [ ] 做 preparation task。
- [ ] 做 Task 1 和 Task 2。
- [ ] 每题写证据：person / task / time / exact sentence。

## 1:40-1:55 TOEFL 桥接

写或说 80 词：

- [ ] 题目：What is the lecture mainly about, and what detail helped you understand it?
- [ ] 必须用 2 个句型：
  - [ ] The lecture is mainly about ...
  - [ ] One important detail is ...
- [ ] 必须包含 1 个原因词：because / so / therefore。
