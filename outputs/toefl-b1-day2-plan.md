## 1:10-1:45 B1 阅读定位

材料：[A conference programme](https://learnenglish.britishcouncil.org/free-resources/reading/b1/conference-programme)

- [ ] 先看标题和版式，不急着读每句。
- [ ] 圈出所有时间、speaker、session title。
- [ ] 做 preparation task。
- [ ] 做 Task 1 和 Task 2。
- [ ] 每道题写答案证据：time / speaker / session title / exact sentence。
- [ ] 错题只改一次，写错因。

- [ ] 必须用 2 个证据词：The programme says ... / This session is about ...
- [ ] 目标 80 词，或 60 秒口头回答。
