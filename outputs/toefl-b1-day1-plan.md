# TOEFL B1 Day 1 学习计划

日期：2026-07-06  
总时长：2 小时  
Day 1 主题：抓主旨，不追求听懂每个词。

## 今日材料

- Sprout：选 1 个 B1 English native-speaker blog/video，主题优先选生活、学习、科技、旅行。
- B1 听力：[British Council - A student discussion](https://learnenglish.britishcouncil.org/free-resources/listening/b1/student-discussion)
- B1 阅读：[British Council - Digital habits across generations](https://learnenglish.britishcouncil.org/free-resources/reading/b1/digital-habits-across-generations)

## 0:00-0:10 热身复习

- [ ] 准备笔记页，写标题：Day 1 - Main Idea。
- [ ] 写下今天目标：I can understand the main idea.
- [ ] 预习 8 个关键词：
  - [ ] compare
  - [ ] similar
  - [ ] different
  - [ ] distance
  - [ ] generation
  - [ ] habit
  - [ ] social media
  - [ ] addiction
- [ ] 每个词口头造 1 个简单句。

## 0:10-0:35 Sprout B1 泛听

- [ ] 选 1 个 B1 native-speaker blog/video。
- [ ] 第一遍不看字幕。
- [ ] 写 3 个关键词。
- [ ] 写 1 句英文主旨：This blog is mainly about ...
- [ ] 给自己打分：听懂大意 0 / 1 / 2 / 3。

## 0:35-0:50 Sprout 精听

- [ ] 第二遍听，可以看字幕。
- [ ] 暂停跟读 5 句。
- [ ] 记录 3 个可复用表达。
- [ ] 选 1 句最有用句子，改成自己的句子。

## 0:50-1:15 B1 听力习题

材料：[A student discussion](https://learnenglish.britishcouncil.org/free-resources/listening/b1/student-discussion)

- [ ] 先做 preparation task。
- [ ] 听第一遍，不看 transcript。
- [ ] 写主旨：Two students are comparing ...
- [ ] 做 Task 1 和 Task 2。
- [ ] 只记录第一次答案分数。
- [ ] 看 transcript，找 3 个表示比较的词或短语。

## 1:15-1:45 B1 阅读习题

材料：[Digital habits across generations](https://learnenglish.britishcouncil.org/free-resources/reading/b1/digital-habits-across-generations)

- [ ] 先看标题，预测文章主题。
- [ ] 3 分钟快速读全文，只抓主旨。
- [ ] 写 1 句主旨：The article is mainly about ...
- [ ] 做 Task 1 和 Task 2。
- [ ] 每道错题标出答案来自哪一句。
- [ ] 收 5 个生词，不超过 5 个。

## 1:45-1:55 TOEFL 桥接

二选一：

- [ ] 口头总结 60 秒：Today I listened to ... I read about ... The main idea was ...
- [ ] 或写 80 词英文总结。

必须包含 3 个连接词：

- [ ] however
- [ ] for example
- [ ] because
